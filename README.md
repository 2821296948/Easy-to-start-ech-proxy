# Easy-to-start-ech-proxy
A lightweight Windows-based proxy launcher that automates the process of starting proxy services and configuring system-wide proxy settings. It ensures seamless proxy management for your network connections, with automatic handling of proxy enablement and disabling upon shutdown.
This is a simplified .bat version developed based on the main program of ech-workers.

一个轻量级的 Windows 平台代理启动器，自动化启动代理服务并配置系统代理设置。在关闭窗口时，它会确保自动禁用系统代理，确保网络连接的无缝管理。
这是基于ech-workers的主程序开发的bat简易版
